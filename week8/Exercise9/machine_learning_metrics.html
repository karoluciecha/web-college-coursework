Types of machine learning metrics

There are quite a few metrics out there to evaluate ML models in different applications. Most of them can be put into two categories based on the types of predictions in ML models.

Classification

Classification is a prediction type used to give the output variable in the form of categories with similar attributes. For example, such models can provide binary output such as sorting spam and non-spam messages.

Some of the popular classification metrics are:

  • accuracy
  • precision
  • recall
  • f1 Score

Regression

Regression is a kind of prediction where the output variable is numerical, not categorical (as opposed to classification). The output is continuous. For example, it can help with predicting a patientâ€™s length of stay in a hospital.

Some of the popular regression metrics are:

  • mse (Mean Squared Error)
  • rmse (Root Mean Squared Error)
  • mae (Mean Absolute Error)

Depending on the use case, using a single metric may not provide you with the complete picture of the problem you are solving. So, you may want to use a few metrics to better evaluate your models.
